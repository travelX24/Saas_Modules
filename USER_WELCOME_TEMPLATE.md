# قالب ترحيب المستخدم الجديد (User Welcome Template)

## كيفية إنشاء القالب

1. اذهب إلى **قوالب البريد الإلكتروني** (Email Templates)
2. اضغط على **إضافة قالب جديد** (Add Template)
3. اختر نوع القالب: **User Welcome** (ترحيب المستخدم)
4. املأ البيانات التالية:

### العنوان (Subject):
```
مرحباً بك في {system_name} — تم تفعيل حسابك بنجاح
```

### المحتوى (Body):
```html
مرحباً {user_name}،

يسرّنا الترحيب بك في {system_name}.
تم تفعيل حسابك بنجاح، وأصبح بإمكانك الآن الدخول والاستفادة من كافة مزايا المنصة لإدارة أعمالك بكفاءة أعلى وتجربة أكثر سلاسة.

<strong>بيانات الدخول:</strong>

<ul>
<li>رابط الدخول: <a href="{login_url}">{login_url}</a></li>
<li>اسم المستخدم: {username}</li>
</ul>

<blockquote>
إن نسيت كلمة المرور، يمكنك إعادة تعيينها من هنا: <a href="{reset_password_url}">{reset_password_url}</a>
</blockquote>

إذا احتجت أي مساعدة، فريق الدعم جاهز لخدمتك:

<ul>
<li>البريد: <a href="mailto:{support_email}">{support_email}</a></li>
<li>الهاتف/واتساب: {support_phone}</li>
<li>ساعات العمل: {support_hours}</li>
</ul>

نشكر لك ثقتك، ونتطلع لأن نكون جزءاً من نجاحك.

مع خالص التحية،
{system_name}
```

## المتغيرات المتاحة

عند اختيار نوع القالب "User Welcome"، ستكون المتغيرات التالية متاحة:

- `{system_name}` - اسم النظام (Athka HR)
- `{user_name}` - اسم المستخدم
- `{login_url}` - رابط الدخول للنظام
- `{username}` - اسم المستخدم أو البريد الإلكتروني
- `{reset_password_url}` - رابط إعادة تعيين كلمة المرور
- `{support_email}` - بريد الدعم
- `{support_phone}` - رقم الدعم
- `{support_hours}` - ساعات العمل
- `{company_name}` - اسم الشركة (إذا كان المستخدم مرتبط بشركة)

## كيفية استخدام القالب

بعد إنشاء القالب، يمكنك:

1. **إرسال يدوي**: اذهب إلى "إرسال بريد إلكتروني" واختر القالب
2. **إرسال تلقائي**: سيتم استخدام القالب تلقائياً عند تفعيل حساب مستخدم جديد (يتطلب إضافة كود في مكان تفعيل الحساب)

## ملاحظات

- يمكنك استخدام HTML في محتوى القالب
- المتغيرات ستُستبدل تلقائياً عند الإرسال
- تأكد من ملء جميع المتغيرات المطلوبة عند الإرسال
