# Athka SaaS Package

SaaS Multi-tenant Package for Laravel - Company Management System

## Installation

### From GitHub Repository

Add the repository to your `composer.json`:
n
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/YOUR_USERNAME/athka-saas.git"
        }
    ],
    "require": {
        "athka/saas": "dev-main"
    }
}
