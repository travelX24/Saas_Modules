# دليل قوالب البريد الإلكتروني (Email Templates Guide)

## أنواع القوالب المتاحة

### 1. انتهاء الاشتراك (Subscription Expiry)
**المتغيرات المتاحة:**
- `{company_name}` - اسم الشركة
- `{expiry_date}` - تاريخ انتهاء الاشتراك
- `{days_remaining}` - عدد الأيام المتبقية
- `{admin_name}` - اسم مدير الشركة

**مثال على العنوان:**
```
تنبيه: انتهاء الاشتراك في {company_name} - {days_remaining} يوم متبقي
```

**مثال على المحتوى:**
```html
مرحباً {admin_name}،

نود إعلامكم بأن اشتراككم في {company_name} سينتهي في تاريخ {expiry_date}.

يتبقى لديكم {days_remaining} يوم لتجديد الاشتراك والاستمرار في الاستفادة من خدماتنا.

ننصحكم بتجديد الاشتراك في أقرب وقت ممكن لتجنب انقطاع الخدمة.
```

---

### 2. الذكرى السنوية للاشتراك (Subscription Anniversary)
**المتغيرات المتاحة:**
- `{company_name}` - اسم الشركة
- `{subscription_start_date}` - تاريخ بداية الاشتراك
- `{years_subscribed}` - عدد سنوات الاشتراك
- `{admin_name}` - اسم مدير الشركة
- `{renewal_date}` - تاريخ التجديد القادم

**مثال على العنوان:**
```
تهنئة: {years_subscribed} سنة من الشراكة مع {company_name}
```

**مثال على المحتوى:**
```html
مرحباً {admin_name}،

يسرنا أن نحتفل معكم بذكرى {years_subscribed} سنة من الشراكة الناجحة مع {company_name}.

لقد بدأتم رحلتكم معنا في {subscription_start_date}، ونحن فخورون بكونكم جزءاً من عائلتنا.

نشكركم على ثقتكم المستمرة ونتمنى لكم المزيد من النجاح والتقدم.

تاريخ التجديد القادم: {renewal_date}
```

---

### 3. إشعارات التحديثات (Update Notification)
**المتغيرات المتاحة:**
- `{company_name}` - اسم الشركة
- `{update_title}` - عنوان التحديث
- `{update_description}` - وصف التحديث
- `{update_version}` - رقم الإصدار
- `{admin_name}` - اسم مدير الشركة
- `{update_date}` - تاريخ التحديث

**مثال على العنوان:**
```
تحديث جديد في النظام - الإصدار {update_version}
```

**مثال على المحتوى:**
```html
مرحباً {admin_name}،

يسرنا إعلامكم بتحديث جديد في النظام بتاريخ {update_date}.

<strong>عنوان التحديث:</strong> {update_title}

<strong>الإصدار:</strong> {update_version}

<strong>الوصف:</strong>
{update_description}

ننصحكم بتحديث النظام للاستفادة من الميزات الجديدة والتحسينات.
```

---

### 4. التهنئة برأس السنة (New Year Greeting)
**المتغيرات المتاحة:**
- `{company_name}` - اسم الشركة
- `{admin_name}` - اسم مدير الشركة
- `{year}` - السنة الجديدة
- `{new_year_date}` - تاريخ رأس السنة
- `{wishes_message}` - رسالة التهنئة المخصصة

**مثال على العنوان:**
```
كل عام وأنتم بخير - سنة {year} سعيدة
```

**مثال على المحتوى:**
```html
مرحباً {admin_name}،

كل عام وأنتم بخير! نتمنى لكم ولشركتكم {company_name} سنة {year} مليئة بالنجاح والازدهار.

{wishes_message}

نشكركم على ثقتكم المستمرة ونتمنى أن تكون السنة الجديدة مليئة بالإنجازات والتقدم.

مع أطيب التمنيات،
فريق {company_name}
```


**مثال على المحتوى:**
```html

{wishes_message}

نتمنى لكم عيداً مباركاً مليئاً بالفرح والبركة.

مع أطيب التمنيات،
فريق {company_name}
```

**المتغيرات المتاحة:**
- `{system_name}` - اسم النظام
- `{user_name}` - اسم المستخدم

- `{reset_password_url}` - رابط إعادة تعيين كلمة المرور
- `{support_email}` - بريد الدعم
- `{support_phone}` - رقم الدعم
- `{support_hours}` - ساعات العمل
- `{company_name}` - اسم الشركة

**راجع ملف `USER_WELCOME_TEMPLATE.md` للتفاصيل الكاملة.**

## كيفية استخدام القوالب

1. **إنشاء القالب:**
   - اذهب إلى "قوالب البريد الإلكتروني"
   - اضغط "إضافة قالب جديد"
   - اختر نوع القالب المناسب
   - املأ العنوان والمحتوى باستخدام المتغيرات المتاحة

2. **إرسال القالب:**
   - اذهب إلى "إرسال بريد إلكتروني"
   - اختر القالب
   - املأ المتغيرات المطلوبة
   - اختر المستلمين (فردي أو متعدد)
   - اختر نوع الإرسال (فوري أو مجدول)

3. **جدولة الإرسال:**
   - يمكنك جدولة الرسائل للإرسال في تاريخ ووقت محدد
   - يمكنك جدولة الرسائل التلقائية (مثل التهنئة السنوية) من خلال الأوامر المجدولة

---

## ملاحظات مهمة

- جميع القوالب تدعم HTML
- المتغيرات تُستبدل تلقائياً عند الإرسال
- يمكنك استخدام المتغيرات في العنوان والمحتوى
- تأكد من ملء جميع المتغيرات المطلوبة عند الإرسال
- يمكنك تخصيص القوالب حسب احتياجاتك

---

## أمثلة على الاستخدام التلقائي

### إرسال تلقائي عند انتهاء الاشتراك:
```php
// في مكان التحقق من انتهاء الاشتراك
$template = EmailTemplate::where('type', 'subscription_expiry')
    ->where('is_active', true)
    ->first();

if ($template) {
    // إرسال الرسالة
}
```

### إرسال تلقائي في رأس السنة:
```php
// في أمر مجدول (Scheduled Command)
$template = EmailTemplate::where('type', 'new_year_greeting')
    ->where('is_active', true)
    ->first();

if ($template) {
    // إرسال الرسالة لجميع الشركات
}
```
